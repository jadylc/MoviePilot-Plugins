# 🎬 MoviePilot-Plugins

MoviePilot第三方插件库，提供了一系列实用的插件来增强MoviePilot的功能。

> ⚠️ 注意：本插件库为个人维护，代码结构参考了其他开源项目。推荐优先使用[官方插件库](https://github.com/jxxghp/MoviePilot-Plugins)。

## 📖 插件详情

### 1. 📊 PT站邀请人统计 (InviterInfo)
- 版本：v1.0.12
- 功能：统计所有PT站的上家信息，包括邀请人信息和邮箱（如果隐私设置允许）
- 标签：PT站
- 特点：
  - 📋 支持多个PT站点的邀请人信息统计
  - 🔍 自动识别站点类型并获取邀请人信息
  - 📅 支持定时任务自动执行
  - 📱 提供可视化的统计数据展示
  - 🔔 支持统计结果通知

### 2. 🏷️ 站点标签&限速 (SiteTagLimit)
- 版本：v1.0.0
- 功能：打标限速一波流，自动为种子添加站点标签并设置上传限速
- 标签：下载器
- 特点：
  - 🏷️ 自动为种子添加站点标签
  - 🚀 支持多下载器管理
  - ⏰ 支持定时任务和固定间隔执行
  - 🎯 支持自定义tracker到站点标签的映射
  - ⚡ 支持基于标签的上传限速设置
  - 🔄 支持覆盖模式和增量模式
- 更新说明：

  <summary>点击查看更新历史</summary>

  </details>

## 📖 使用说明

1. 在MoviePilot中安装插件
2. 根据插件说明配置相关参数
3. 启用插件并设置定时任务（如需要）
